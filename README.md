Project Title
Menu Driven Array Analysis Program in C++

What the Program Does
This program performs basic analysis operations on an integer array using a menu-driven command-line interface.
It demonstrates core programming fundamentals such as arrays, loops, conditionals, and functions.

Features

Display array elements
Calculate sum and average
Find maximum and minimum values
Count even and odd numbers
Count positive, negative, and zero values
Reverse the array
Search an element
Find second largest element
Left rotate array by 1
Right rotate array by 1
Menu-driven execution

How to Run

Compile:
g++ array_analysis.cpp -o array_analysis

Execute:
./array_analysis
